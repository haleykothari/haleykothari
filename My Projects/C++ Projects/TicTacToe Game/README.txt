TicTacToeGame.cpp

User can play a Tic Tac Toe game with the computer 

Code contains a 2D array that covers both the rows and the columns. 
User will play with computer until one wins and the code will tell the user whether they win or lose

User is the X and the Computer is O

The code includes a draw conclusion and will output that is needed.
Include a temporary "~" as a marker to ensure that the computer stays on grid and to make sure the 
computer does not marker on a spot that is already taken. If there is no "~" in a section of the grid the computer 
will mark the next grid spot that has a "~".
